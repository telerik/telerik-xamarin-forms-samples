Overview

RichTextEditor is a control that is able to display and edit rich-text content.

The control allows you to edit text and apply rich formatting options, like:

  • Bold, Italic
  • Underline, Strike Through, Superscript and Subscript
  • Text color and background
  • Bullet and numbered lists
  • Paragraph alignment
